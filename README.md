# E-Commerce Platform

## Overview
This is a full-stack e-commerce platform built using Python and Flask. The project demonstrates backend order management, analytics dashboards, and automation scripts designed to optimize store operations. It highlights full-stack development, data handling, and business process optimization skills.

## Features
- Product catalog and inventory management
- Shopping cart and checkout functionality
- Backend order processing and reporting
- Analytics dashboards for operational insights
- Automation scripts for data processing and reporting

## Tech Stack
- **Backend:** Python, Flask, SQLAlchemy
- **Frontend:** HTML, CSS, Jinja2 templates
- **Data & Analytics:** Pandas, Openpyxl
- **Other:** Regex for validation, CSV export functionality
